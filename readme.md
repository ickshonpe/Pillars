## Controls
<br>

### Keyboard
 Input | Action
:---------------:|:------------:
   Space         | Start \ Pause \ Unpause  
   Z             | Cycle Up
   X             | Cycle Down
   Left Arrow    | Move left
   Right Arrow   | Move Right
   Down Arrow    | Quick Fall      
   Escape        | Quit

<br>
<br>

### Controller
 Input | Action
:-------------:|:------------:
   Start       | Start \ Pause \ Unpause  
   B           | Cycle Up
   A           | Cycle Down
   Left Stick  | Move
